\section{Colours}

Colours are available through: \\
\verb!solin-colour1, ..., solin-colour7!,\\
\verb!light-colour1, ..., light-colour7!,\\
\verb!wong-colour1, ..., wong-colour7!, and\\
\verb!tol-colour1, ..., tol-colour7!.

In addition, one can access the colours using \verb!colour1, ..., colour7! which defaults to the Solin colour palette. This can be changed by changing the argument of the \verb!\usepackage[arg]{safecolours}! to \verb!light!, \verb!wong! or \verb!tol!.
\\[1em]
