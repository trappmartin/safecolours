\section{Colours}

Colours are available through: \textcolor{scBlue}{\texttt{scBlue}} (blue), \textcolor{scGreen}{\texttt{scGreen}} (green), \textcolor{scRed}{\texttt{scRed}} (red), \textcolor{scPurple}{\texttt{scPurple}} (purple), \textcolor{scYellow}{\texttt{scYellow}} (yellow), \textcolor{scCyan}{\texttt{scCyan}} (cyan), and \textcolor{scGrey}{\texttt{scGrey}} (grey).

The default colour palette is Tol. This can be changed by passing an option to the package, e.g., \verb!\usepackage[light]{safecolours}! for the Light colour palette.

Alternatively, one can access the individual colour palettes directly using:\\
\verb!solin-colour1, ..., solin-colour7!,\\
\verb!light-colour1, ..., light-colour7!,\\
\verb!wong-colour1, ..., wong-colour7!, and\\
\verb!tol-colour1, ..., tol-colour7!.
\\[1em]
